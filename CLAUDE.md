# EPPA — Claude Workflow Rules

## Git Workflow

- **Before making any changes**, create a new git branch from main. Name it descriptively (e.g. `feature/solo-logic`, `fix/midi-filter`, `docs/readme`).
- **After completing each user prompt**, commit all changes on the current branch with a clear commit message.
- **Never commit directly to main.**
- **When the user says "merge"**, merge the current feature branch into main.
